# Panoramica di [!DNL Quality Patches Tool] (QPT) v1.0.18

Questa sottosezione fornisce una descrizione dettagliata dei problemi risolti dalle patch disponibili in [!DNL Quality Patches Tool] (QPT) v1.0.18.

QPT v1.0.18 include le seguenti patch:

1. **MDVA-32655**: lo stato del messaggio *in corso* non corretto è stato corretto nel messaggio *completo* per il consumatore `quoteItemCleaner` dopo l&#39;eliminazione di diversi prodotti.
1. **MDVA-34102**: è stato risolto il problema per cui la quantità di scorte predefinite è zero per i prodotti disabilitati nelle pagine [!UICONTROL Product Grid] e [!UICONTROL Edit Product] dell&#39;amministratore.
1. **MDVA-34189**: corregge il timeout di 503 primi byte nelle query [!DNL Visual Merchandiser] durante il caricamento della pagina categoria amministratore.
1. **MDVA-34695**: dopo l&#39;eliminazione delle categorie, vengono corretti `children_count` negativi.
1. **MDVA-35286**: risolve il problema relativo alla presenza di un errore se un cliente ha raggruppato i prodotti nel carrello e passa dall&#39;estrazione di più indirizzi all&#39;estrazione nella pagina.
1. **MDVA-35312**: è stato risolto il problema che causava la generazione del codice di risposta di errore 500 invece del codice 200 da parte di una richiesta GraphQL vuota

Utilizza il menu a sinistra per passare a una pagina patch specifica.
